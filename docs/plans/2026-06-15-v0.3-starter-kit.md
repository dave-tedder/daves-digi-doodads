# v0.3 Starter Kit Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make `daves-digi-doodads` easier for non-technical friends to use as a full bundle or as cherry-picked skills, rules, and templates across Claude Code, Codex, Antigravity, and similar agent tools.

**Architecture:** Keep `goof-proofs` as the full Claude Code plugin bundle. Add a tool-neutral starter layer around it: a short front-door doc, sanitized templates, cross-tool setup docs, and cherry-pick recipes. Do not publish Dave's private identity block, account inventory, business configuration, Notion details, MCP URLs, API keys, or other sensitive setup.

**Tech Stack:** Markdown docs, Claude Code plugin manifests, portable `SKILL.md` directories, Codex `AGENTS.md`, Claude Code `CLAUDE.md`, git.
