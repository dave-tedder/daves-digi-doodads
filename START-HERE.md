# Start Here

This repo is a public starter kit for agent workflows: skills, project tracking templates, and setup notes that came from real project work.

## Pick Your Path

### I Use Claude Code

Install the `goof-proofs` plugin if you want the full bundle.

### I Use Codex

Use the templates under `templates/` and copy selected skills into `~/.codex/skills/`.

### I Use Antigravity Or Another Agent

Use the portable `SKILL.md` folders and the markdown templates.

### I Only Want Project Tracking

Start with the project tracker and session log templates.

## What Not To Copy

Do not copy someone else's private identity block, account inventory, MCP URLs, API keys, business database IDs, or personal operating rules without reviewing them.

## Brain Bank

When Brain Bank is public, this repo can act as the companion setup kit: project tracking, session closeout, skill and rule triage, and memory capture habits.
